# Proyecto: Taxes Calculator

Este proyecto implementa una calculadora de impuestos en Java que determina el importe a pagar según el ingreso anual, la categoría fiscal y el estado civil del contribuyente.


## Funcionalidad del código
 
- Cálculo de impuestos para dos categorías:
  - **Categoría A**: aplica tramos progresivos segun el ingreso que tenga el contribuyente.
  - **Categoría B**: aplica un porcentaje fijo al ingreso que reciba el contribuyente.
- Hay un descuento adicional si el contribuyente está casado.
- Uso de una clase independiente (`TaxConstants`) para almacenar las constantes que se utilizan en el codigo.
- Código documentado completamente con **JavaDoc**.
